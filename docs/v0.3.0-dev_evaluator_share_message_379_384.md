## What It Is

Quant Strategy Lab is a local-first desktop research platform for:

- Importing OHLCV market data from CSV/TXT files.
- Building and running quantitative strategy candidates.
- Running backtests and validation workflows.
- Inspecting ranking, trades, equity, and strategy details.
- Exporting research reports.
- Testing archive import preview behavior.

This is a developer pre-release, not a public production release.

## How To Run

1. Download the zip.
2. Unzip it to a normal folder.
3. Open the extracted folder.
4. Double-click `QuantStrategyLab.exe`.
5. Create a project with `New Project`.
6. Import your own OHLCV CSV/TXT data, or browse to bundled samples under:

```text
_internal/sample_data/
```
